# ECL Material Requirement Planner

Three-page Streamlit app for factory material planning.

## Pages

1. Required Components
2. Stock Entry
3. Final Summary
4. Product List

## Deployment

Upload to GitHub and deploy on Streamlit Community Cloud.

Main file path:

app.py

## Required files

- app.py
- requirements.txt
- runtime.txt
- README.md
- .streamlit/config.toml
